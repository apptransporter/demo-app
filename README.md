### Getting started

Create environment config

    cp .env-dist .env

Build images

    docker-compose build
    
Inital setup of application    

    docker-compose run --rm -e APP_ADMIN_PASSWORD=admin1 php yii app/setup
        
Start stack
        
    docker-compose up -d
